Detailed Description

CSV parser that acts on a file as input.

Note
Some part of this API documentation refers to following standards:
  • CSV-1203 available here: http://mastpoint.com/csv-1203
  • RFC 4180 available here: https://tools.ietf.org/html/rfc4180
