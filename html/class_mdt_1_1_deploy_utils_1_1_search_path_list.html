void Mdt::DeployUtils::SearchPathList::setPathSuffixList ( const QStringList &  suffixList)

Specify subdirectories to check.

Specify subdirectories to check below each directory in those specified with setPathPrefixList(). For example, if /opt/liba and /opt/libb have been set as path prefixes, and suffixList contains bin qt5/bin, path list will contain /opt/liba/bin, /opt/liba/qt5/bin, /opt/libb/bin, /opt/libb/qt5/bin . If includePathPrefixes is set, path list will contain /opt/liba, /opt/liba/bin, /opt/liba/qt5/bin, /opt/libb, /opt/libb/bin, /opt/libb/qt5/bin .
